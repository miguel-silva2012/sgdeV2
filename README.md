## descrição

Um projeto de sistema de gerenciamento de estoque para uma empresa pequena cujas tecnologias utilizadas foram java, javaFx, SQL(MySQL) e css para a aparência do App.

### pré-requisitos

A linguagem de programação usada neste projeto foi o Java 21 (LTS) com a biblioteca JavaFx, o projeto usa SQL(MySQL) para guardar dados com mais segurança. Então serão necessários:

- **Java(21)**;
- **JavaFx**;
